# **C-Reference**
C语言学习指南，取自[CPPReference](cppreference.com)，个人重写部分示例代码。

本人所使用的 GCC 版本为 10.3.0 。

为了保证初学者可以在任何环境下进行编译执行，附加了编译指令以及执行指令。在源代码目录下打开命令行，执行编译指令即可编译源文件。
